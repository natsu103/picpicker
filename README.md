# PicPicker

一个简洁高效的 Android 照片浏览和管理应用，通过直观的手势操作帮助用户快速整理照片库。

## 功能特性

### 🎯 核心功能
- **随机浏览模式**：打乱时间线，每次滑动都是惊喜
- **批量浏览模式**：随机抽取20张照片为一组，轻快便捷
- **收藏管理**：创建个人收藏夹，保存喜欢的照片
- **照片删除**：标记待删除照片，批量确认删除

### ✨ 手势操作
- **上滑**：标记为待删除，浏览结束后统一确认
- **下滑**：收藏该照片，自动跳到下一张
- **左右滑动**：切换浏览上一张/下一张照片
- **长按**：查看照片的拍摄时间、位置等详细信息

### 📱 界面特色
- 沉浸式全屏浏览体验
- 黑色状态栏设计
- 简洁直观的用户界面
- 流畅的动画效果

## 技术架构

### 开发环境
- **编译版本**：Android SDK 34
- **最低版本**：Android 6.0 (API 26)
- **开发语言**：Java
- **构建工具**：Gradle

### 主要依赖
- AndroidX 组件库

## 快速开始

### 环境要求
- Android Studio Arctic Fox 或更高版本
- Android SDK 34
- Java 11

### 构建步骤
1. 克隆项目到本地
2. 使用 Android Studio 打开项目
3. 同步 Gradle 依赖
4. 连接 Android 设备或启动模拟器
5. 点击运行按钮构建并安装应用

### 权限说明
应用需要以下权限：
- 读取外部存储权限
- 写入外部存储权限
- 读取媒体图片权限
- 管理外部存储权限

## 使用指南

### 首次使用
1. 启动应用后授予必要的存储权限
2. 在主页面选择浏览模式：
   - **开始浏览**：完整浏览所有照片
   - **一组浏览**：随机20张照片为一组
   - **我的收藏**：查看已收藏的照片

### 手势操作说明
- **上滑删除**：将当前照片标记为待删除
- **下滑收藏**：将照片添加到收藏夹
- **左右滑动**：浏览相邻照片
- **长按查看**：显示照片详细信息

### 批量删除流程
1. 在浏览过程中上滑标记要删除的照片
2. 浏览结束后点击"确认删除"按钮
3. 在确认界面查看待删除照片列表
4. 确认删除操作（无法撤销）

## 开发说明

### 核心类说明

#### MainActivity
- 照片浏览的核心界面
- 实现手势识别和照片切换逻辑
- 管理删除列表和收藏列表

#### SwipeInterceptorLayout
- 自定义手势拦截布局
- 识别上下左右滑动方向
- 提供手势回调接口

#### PhotoAdapter
- 照片列表适配器
- 处理图片加载和显示
- 支持照片信息提取（时间、位置等）

### 数据存储
- 使用 SharedPreferences 存储收藏列表
- 照片数据通过 ContentResolver 从媒体库获取
- 删除操作直接操作媒体库
